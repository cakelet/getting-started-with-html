UNIT 1

World Wide Web

The WWW is a collection of html (hypertext markup language) documents.

Browsers -- the Internet -- Servers
Browsers request information of the Internet via servers using http (main protocol of the web).

HTML

Made up of: text content; markup; references; links

Markup
Tells the computer how to display text.
Made up of tags
[tag name]Content[/tag name] - Whole phrase is called an element.

Attributes
References to other files - web pages, images, videos, etc.
Tags can have multiple attributes.
Anchor tag (link): [a href="www.website.com"]display text[/a]
Image tag: [img src="url" alt="display text"]

Image tags have no closing tag - Void tag

White Space
Space created through br (inline, void) or p (block, not void) tags.

Container elements
Used to attach style for display. Span or div. Can also be inline or block.

Inline - applies to that line only.
Block - creates a "box" of space that can have width and length.
